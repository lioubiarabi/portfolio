EasyColoc

Roommate expense management application focused on shared living organization, debt calculation, and admin supervision.

Stack: Laravel, PHP, MySQL, Tailwind CSS
LaravelPHPMySQLTailwind

Overview

This project helps roommates manage shared expenses, invitations, debts, and trust indicators inside one platform. It combines business logic, dashboard views, and data relationships in a practical Laravel application.

Highlights

Automatic debt calculation, invitation token flow, reputation features, and admin dashboard management.
