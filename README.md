# Loubi Arabi Portfolio

Modern, professional single-page portfolio website.

## Highlights

- Neon-inspired hero section based on your reference style and profile image.
- Projects section with 16 featured repositories and category filters.
- Certificates section with 24 listed credentials and issuer filters.
- Responsive navigation, dark/light theme, smooth scrolling, and reveal effects.

## Run locally

```bash
python3 -m http.server 8080
```

Then open `http://localhost:8080`.
