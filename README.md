# Loubi Arabi Portfolio

A professional and modern single-page portfolio website highlighting:

- Hero introduction with profile image and performance metrics.
- About, expertise, and service sections.
- Filterable project showcase.
- Contact call-to-action and CV access.
- Responsive and theme-aware interface.

## Run locally

Open `index.html` directly in a browser, or use a static server:

```bash
python3 -m http.server 8080
```
